Google Chrome

Google Chrome adalah peramban web yang dikembangkan oleh Google, dirilis pada tahun 2008. Chrome adalah peramban web terpopuler di dunia saat ini!

Mozilla Firefox

Mozilla Firefox adalah peramban web sumber terbuka yang dikembangkan oleh Mozilla. Firefox telah menjadi peramban web terpopuler kedua sejak Januari 2018.

Microsoft Edge

Microsoft Edge adalah peramban web yang dikembangkan oleh Microsoft, dirilis pada tahun 2015. Microsoft Edge menggantikan Internet Explorer.
